#  Cyberbot – Cybersecurity Awareness Chatbot

##  Project Overview
Cyberbot is a C# console-based chatbot designed to educate South African citizens about cybersecurity threats such as phishing, malware, safe password practices, and safe browsing habits.  

This project was developed as part of a programming assignment to demonstrate knowledge of object-oriented programming, user interaction, input validation, multimedia integration, and GitHub version control.

---

##  Purpose of the Project
The purpose of Cyberbot is to:
- Raise awareness about cybersecurity threats
- Educate users on safe online behaviour
- Simulate real-life cyber threat scenarios
- Provide interactive learning through a chatbot interface

---

##  Features

###  Voice Greeting
- Plays a welcome message when the program starts using a WAV file.
- Adds a more interactive user experience.

###  ASCII Art Interface
- Displays a cybersecurity-themed logo on startup.
- Improves visual presentation in the console.

###  Personalised User Interaction
- Asks for the user’s name.
- Provides personalised chatbot responses.

### Chatbot Responses
Cyberbot can respond to topics such as:
- Phishing attacks
- Password safety
- Safe browsing
- General cybersecurity questions

###  Input Validation
- Detects empty or invalid inputs.
- Provides a default response when input is not recognised.

###  Enhanced Console UI
- Uses colours, spacing, and borders.
- Improves readability and user experience.
